# Проект Mesto: карточки и попапы

### Обзор
**Сайт с адаптивной вёрсткой от 320px до 1280px и выше.**
**Стилизация сайта реализована при помощи CSS.**
**Создано 3 окна PopUp, в 2х формах PopUp реализован ввод данных с сохранением в структуру проекта, в 3м PopUp реализована функция добавления карточек и лайков. Все 3 окна имеют кнопку закрытия. Весь функционал создан при помощи Java Script.**
